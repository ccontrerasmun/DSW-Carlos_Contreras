Maquetación

Etiquetas

Etiquetas Descripción
head Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo
title Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página
base Define la URL base para las URLs relativas en la página
Link Usada para enlazar JavaScript y CSS externos con el documento HTML actual
style Etiqueta de estilo usada para escribir CSS en línea
boby Representa el contenido principal de un documento HTML. Solo hay un elemento en un documento
address Define una sección que contiene información de contacto
p Define una parte que debe mostrarse como un párrafo
hr Representa un quiebre temático entre parrafos de una sección o articulo o cualquier contenido
dl Define una lista de definiciones, es decir, una lista de términos y sus definiciones asociadas


Atributos

Atributos Descripción
accept Lista de tipos aceptados por el servidor, usualmente un tipo de archivo
Action La URL del programa que procesa la información enviada en el formulario
aling Especifica el alineamiento horizontal del elemento
alt Texto alternativo en caso de que la imagen no se pueda mostrar
autofocus El elemento deberia recibir foco automaticamente después de haberse cargado la página
autoplay El audio o video deberia empezar lo antes posible
challenge Cadena de desafio que se envia junto con la clave pública
controls Indica si el navegador debe mostrar controles de reproducción al usuario
data Especifica la URL del recurso
dir Define una lista de definiciones, es decir, una lista de términos y sus definiciones asociadas
